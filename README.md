# CoffeeNet - سامانه کافی‌شری

سامانه کامل و حرفه‌ای کافی‌شری با استفاده از HTML، TailwindCSS و JavaScript.

## ویژگی‌های سامانه

- کاملاً فرانت‌اند و بدون نیاز به بک‌اند
- استفاده از LocalStorage برای ذخیره‌سازی داده‌ها
- طراحی واکنش‌گرا و زیبا با TailwindCSS
- فونت فارسی (Vazirmatn) و پشتیبانی کامل از راست به چپ
- سیستم مدیریت سفارش چندمرحله‌ای
- پنل مشتری و پنل مدیریت
- کامپوننت‌های قابل استفاده مجدد
- اعتبارسنجی فرم‌ها
- چاپ فاکتور با قالب اختصاصی

## ویژگی‌های کلیدی

### کامپوننت‌های مشترک
- Header و Footer
- مدال‌ها و نوتیفیکیشن‌ها
- جداول و فرم‌ها
- کامپوننت‌های پیشرفته (Stepper, Tabs, Accordion)

### صفحات عمومی
- صفحه اصلی با بخش‌های مختلف
- لیست خدمات با قابلیت جستجو و فیلتر
- صفحه جزئیات خدمات
- سیستم ثبت سفارش چندمرحله‌ای

### پنل مشتری
- ورود با OTP
- داشبورد شخصی
- لیست سفارش‌ها
- پیگیری وضعیت سفارش

### پنل مدیریت
- داشبورد با KPIها
- مدیریت سفارش‌ها
- مدیریت خدمات
- گزارش‌های مالی
- تنظیمات سیستم

## نحوه استفاده

برای اجرای سامانه کافی است فایل `index.html` را در مرورگر خود باز کنید.

داده‌های سامانه در LocalStorage مرورگر ذخیره می‌شوند.

## اعتبارات

- طراحی با TailwindCSS
- فونت فارسی: Vazirmatn
- آیکون‌ها: Font Awesome
